# BatchPulse-Continuous-Batching-and-Scheduler-Profiler-for-vLLM
A serving sandbox that studies how request scheduling affects TTFT, inter-token latency, and throughput. Include experiments for mixed prompt lengths, bursty traffic, and different max_num_seqs / max_num_batched_tokens settings, with a dashboard that shows where batching helps and where it hurts.
